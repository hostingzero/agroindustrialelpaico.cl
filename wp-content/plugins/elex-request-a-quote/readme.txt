== Description ==
Create Request a Quote option for your WooCommerce products. You can also create and customize request a quote forms to be displayed on the frontend. The plugin will also send automated email notifications for quote submissions, approvals, and rejections.

= Features =
* Easily add Request a Quote option for WooCommerce Products.
* Supports both Simple and Variable Products.
* Easily hide the Add to Cart button.
* Customize WooCommerce Request a Quote from the way you want.
* Trigger Email notifications to the Customers and Admin.

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. That's it ! you can now configure the plugin.
